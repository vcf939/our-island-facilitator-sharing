推廣回饋與案例共學表

這份表單用來收集老師實際推廣《我們的島》桌遊、使用技能包後的經驗。重點不是評分，而是把現場發生了什麼、哪裡有效、哪裡卡住整理出來，成為下一位老師可以參考的共學材料。

本回饋表為引導師 Vincent 個人共學整理使用，非至善基金會官方表單。請避免填寫可辨識學生個資。

正式回饋表已建立

老師可直接填寫 Google Form。下方題目稿與腳本保留作為日後調整表單、複製新版或共備說明使用。

填寫推廣回饋
下載 Google Form 題目稿 下載回覆欄位 CSV 下載自動建表腳本 下載案例牆表單腳本
區塊 1

填寫者與場域

  1. 填寫者姓名或暱稱
  2. 服務單位或場域類型：國小、國中、高中職、大專、社福、社區、其他
  3. 縣市或地區
  4. 身份：導師、科任/領域教師、輔導老師、社工、行政、其他
  5. 是否願意後續被聯繫：願意 / 暫不 / 只願意匿名分享
  6. 聯絡方式，選填
區塊 2

這次如何使用

  1. 使用日期與活動長度
  2. 參與對象與年齡層
  3. 參與人數
  4. 使用版本：完整體驗、短版體驗、議題討論版、教師共備版、其他
  5. 使用了哪些技能包：開場安全聲明、ORID 題庫、敏感情境回應卡、校園情境清單、情緒地圖、副作用與修復檢核、其他
  6. 活動目標：反歧視理解、班級經營、霸凌預防、生命教育、公民教育、輔導活動、其他
區塊 3

現場發生了什麼

  1. 最有反應的一刻是什麼？
  2. 有沒有一句讓你印象深刻的學生/參與者發言？
  3. 參與者比較容易進入的是哪一段：角色、線索、結局、沉澱、校園情境、行動承諾？
  4. 現場出現哪些情緒：好奇、興奮、尷尬、防衛、沉默、憤怒、難過、理解、其他？
  5. 你如何回應這些情緒或衝突？
區塊 4

卡點與風險

  1. 這次最大的困難是什麼？時間、規則、班級氣氛、敏感議題、學生衝突、師長觀點、資源不足、其他
  2. 是否出現歧視、嘲笑、貼標籤或霸凌相關片段？你怎麼處理？
  3. 是否有參與者感到受傷、沉默、退出或過度激動？後續如何照顧？
  4. 如果再帶一次，你會提前做什麼安全設計？
區塊 5

技能包使用回饋

  1. 哪些工具最有幫助？為什麼？
  2. 哪些工具不夠清楚或不容易使用？
  3. 你希望增加什麼：更多校園情境、年段版本、家長溝通、行政說明、短版流程、其他？
  4. 這份技能包是否提升你推廣的意願：1 到 5 分
  5. 這份技能包是否提升你實際帶領的信心：1 到 5 分
區塊 6

案例分享同意

  1. 是否願意將你的經驗整理成案例牆：願意具名 / 願意匿名 / 暫不公開
  2. 若公開，請寫下你希望保留或隱去的資訊
  3. 一句話給下一位想推廣的老師
  4. 下一步你想做什麼：再帶一次、找同伴共備、改成班級課程、邀請社工/輔導合作、先暫停整理

表單設定建議

不強制收真名；聯絡方式設為選填；案例公開前再二次確認。

隱私提醒

不要收學生姓名、班級座號、照片或可辨識事件細節。

整理節奏

每月挑 1 到 3 則，改寫成「情境、做法、學習」三段式案例。

回到共學

回饋不是成果比賽，而是幫下一位老師少一點孤單、多一點方法。
