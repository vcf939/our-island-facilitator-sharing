歧視事件情緒地圖

使用目的

歧視事件不只是「誰說了什麼」，也是一連串情緒互相碰撞。

老師若只處理對錯，容易讓被歧視者覺得沒有被接住，也容易讓造成傷害的人停在防衛。這份情緒地圖幫助老師先看懂情緒，再把現場帶回覺察、責任與修復。

一、被歧視者可能出現的情緒

1. 受傷與委屈

可能想法：

  • 為什麼又是我？
  • 我明明沒有做錯什麼。
  • 為什麼大家都覺得這很好笑？

老師可注意：

  • 學生是否突然安靜、低頭、離開現場。
  • 學生是否用笑掩飾不舒服。
  • 學生是否說「沒關係」但表情明顯不自然。

2. 羞恥與尷尬

可能想法：

  • 我被大家看見了。
  • 我的家庭、身份或身體被公開討論。
  • 我好想消失。

老師可注意：

  • 是否有學生被迫公開族群、家庭、資源班、身心狀況或經濟困難。
  • 是否有學生因被點名而不再參與。

3. 憤怒

可能想法：

  • 這不公平。
  • 你們憑什麼這樣說我？
  • 我已經忍很久了。

老師可注意：

  • 憤怒不一定是問題本身，可能是受傷後的保護反應。
  • 不要只處理回嘴或爆發，要回頭看前面發生了什麼。

4. 害怕與焦慮

可能想法：

  • 如果我說出來，會不會被更多人笑？
  • 老師會相信我嗎？
  • 下次會不會又發生？

老師可注意：

  • 學生是否擔心報復、被孤立或被貼標籤。
  • 是否需要私下談，而不是在全班面前追問。

5. 孤單與被排除

可能想法：

  • 沒有人站在我這邊。
  • 我不是這個班的一份子。
  • 我跟大家之間的橋斷掉了。

老師可注意：

  • 分組、座位、群組互動是否讓學生長期被排除。
  • 旁觀者是否沉默或跟著笑。

6. 困惑與自我懷疑

可能想法：

  • 是不是我太敏感？
  • 是不是我真的很奇怪？
  • 是不是這種玩笑本來就要忍？

老師可注意：

  • 不要用「你不要想太多」回應。
  • 可以先確認感受：「你覺得不舒服，這件事值得被認真看待。」

7. 麻木與放棄

可能想法：

  • 說了也沒有用。
  • 反正每次都這樣。
  • 我不要再期待有人理解。

老師可注意：

  • 表面沒反應不代表沒受傷。
  • 長期沉默可能是重複經驗後的保護方式。

二、造成傷害者可能出現的情緒

1. 好玩與得意

可能想法：

  • 大家都笑了，代表我很好笑。
  • 我只是炒熱氣氛。

老師可引導：

  • 笑聲不一定代表沒有人受傷。
  • 幽默可以存在，但不需要建立在貶低別人上。

2. 優越感

可能想法：

  • 我比較正常。
  • 他們本來就比較奇怪。
  • 這種人就是這樣。

老師可引導：

  • 把「這種人」改成「這個人」。
  • 看見每個人不是身份標籤的總和。

3. 不耐煩

可能想法：

  • 幹嘛這麼認真？
  • 現在什麼都不能講了嗎？

老師可引導：

  • 不是什麼都不能講，而是練習看見話語的影響。
  • 可以討論界線，不是直接禁止所有玩笑。

4. 防衛與委屈

可能想法：

  • 我又不是故意的。
  • 為什麼都怪我？
  • 我被誤會了。

老師可引導：

  • 意圖和影響可以分開看。
  • 你可能沒有想傷害人，但仍然可以為造成的影響負責。

5. 羞愧與尷尬

可能想法：

  • 我是不是很糟？
  • 大家會不會覺得我是壞人？

老師可引導：

  • 承認造成影響，不等於你整個人都是壞的。
  • 修復比否認更能讓關係往前走。

6. 害怕

可能想法：

  • 我會不會被處罰？
  • 我會不會被貼上歧視者標籤？

老師可引導：

  • 現在的重點是理解、負責與修復。
  • 如果只害怕處罰，學不到怎麼下次做得更好。

7. 困惑

可能想法：

  • 我真的不知道這句話會傷人。
  • 以前大家都這樣說。

老師可引導：

  • 不知道可以成為學習的開始。
  • 我們一起看：這句話連到什麼刻板印象？對方可能承受了什麼長期經驗？

三、旁觀者可能出現的情緒

1. 想笑又不安

可能想法：

  • 大家都笑，我不笑好像很怪。
  • 但我又覺得哪裡不太對。

2. 害怕被牽連

可能想法：

  • 如果我出聲，下一個會不會變成我？
  • 我不想破壞朋友關係。

3. 道德焦慮

可能想法：

  • 我知道不對，但不知道怎麼阻止。
  • 我怕說錯話。

4. 麻木與習慣

可能想法：

  • 班上本來就這樣。
  • 反正每次都有人被開玩笑。

四、最常見的情緒落差

| 造成傷害者的感受 | 被歧視者的感受 | 現場衝突 |
|---|---|---|
| 我只是開玩笑 | 我被羞辱了 | 意圖與影響不同 |
| 我又不是故意 | 但我真的受傷 | 無意不等於無影響 |
| 你太敏感 | 我已經忍很久 | 單次事件與長期累積不同 |
| 我被誤會了 | 我不被相信 | 雙方都覺得自己受委屈 |
| 這沒什麼 | 這對我很重要 | 經驗位置不同 |

五、老師可以使用的引導句

接住被歧視者

  • 謝謝你說出來，這件事值得被認真看待。
  • 你不需要現在說出所有細節，我們可以先確認你需不需要休息。
  • 你覺得不舒服，不代表你太敏感。

降低造成傷害者的防衛

  • 我知道你可能沒有想傷害人，我們現在一起看的是影響。
  • 承認這句話造成影響，不代表你整個人都是壞的。
  • 你可以先不用急著解釋，我們先聽聽對方感受到什麼。

支持旁觀者

  • 如果你剛剛不知道怎麼反應，這很正常。我們可以一起練習下一次怎麼做。
  • 旁觀者不一定要一次說很大的話，有時候一句「這樣說不太舒服」就是開始。
  • 班級文化不是只有老師建立的，也跟每個旁觀者的選擇有關。

六、用在《我們的島》的提醒

遊戲中的角色情緒不一定等於真實學生情緒，但可能喚起真實經驗。

老師可以在卸角後問：

  • 剛剛你在角色中有什麼感覺？
  • 這個感覺有沒有讓你想到現實生活中的某些時刻？
  • 如果類似情況發生在班上，我們希望彼此怎麼回應？
