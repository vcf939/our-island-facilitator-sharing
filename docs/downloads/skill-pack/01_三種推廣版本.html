Skill Pack

三種推廣版本

使用目的

讓老師知道《我們的島》不只有一種推廣方式。

如果老師一開始就覺得必須完整執行所有課程，容易因為時間、備課與信心不足而延後。三種版本能讓老師依照自己的場域，選擇一個可開始的入口。

A. 15-20 分鐘輕量版

適合場域

  • 班會
  • 導師時間
  • 綜合活動課暖身
  • 社團開場
  • 教師共備引言
  • 生命教育或公民課前導討論

核心目標

先不完整帶遊戲，而是用一個角色、一段故事或一句歧視語言，引發學生對「玩笑、刻板印象與傷害」的覺察。

建議流程

  1. 故事引入，3 分鐘
    簡述依闊群島、彩虹族、繪畫比賽與酸言酸語事件。
  2. 一句話討論，5 分鐘
    選一句類似「彩虹族不是都很會畫畫嗎？」的話，請學生判斷：這是玩笑、刻板印象，還是傷害？
  3. 角色換位，5 分鐘
    問學生：如果你是被這句話說中的人，你可能有什麼感覺？如果你是旁邊的人，你可以怎麼做？
  4. 小行動收束，3-5 分鐘
    請學生寫下一句「下次我可以怎麼說得更尊重」。

適合老師

  • 還沒有把握完整帶遊戲。
  • 班級時間零碎。
  • 想先測試學生對議題的反應。
  • 想用低風險方式開啟對話。

B. 70-90 分鐘標準版

適合場域

  • 兩節連堂
  • 綜合活動
  • 輔導課
  • 公民/社會課
  • 主題週工作坊
  • 學生營隊

核心目標

讓學生完整體驗遊戲情境，透過角色、討論、交流、判定與結局，理解歧視如何發生，並在遊戲後進行反思。

建議流程

  1. 開場與安全聲明，5 分鐘
  2. 情境帶入與角色閱讀，10 分鐘
  3. 第一回合討論、交流、報告、判定，20 分鐘
  4. 第二回合線索與再次討論，20 分鐘
  5. 結局公布與卸角，5 分鐘
  6. 遊戲秘密與角色隱喻，10 分鐘
  7. ORID 反思，15-20 分鐘

適合老師

  • 已經參與過工作坊。
  • 有兩節連堂或完整活動時間。
  • 願意讓學生透過體驗而非講授進入議題。

C. 三階段延伸版

適合場域

  • 主題週
  • 跨領域課程
  • 班級經營系列課
  • 生命教育/人權教育方案
  • 校園友善文化推動

核心目標

讓學生不只玩過遊戲，也能從體驗、反思走到具體行動。

建議流程

  1. 玩吧！遊戲體驗，70-90 分鐘
    讓學生進入角色與情境。
  2. 想吧！故事背後，45 分鐘
    從現實連結、換位同理、刺激思考三個方向選一個深入討論。
  3. 走吧！展開行動，45 分鐘
    設計班級行動、個人行動或校園友善倡議。

適合老師

  • 想把桌遊納入完整教案。
  • 想結合班級經營或校園文化。
  • 有機會和同事、輔導室或行政合作。

選擇版本的判斷問題

請老師先回答：

  1. 我現在有多少時間？
  2. 我的學生對討論敏感議題熟悉嗎？
  3. 我的班級是否需要先建立安全感？
  4. 我是否有協同老師或輔導資源？
  5. 我希望這次先做到「引發覺察」、「完整體驗」，還是「帶出行動」？

給老師的提醒

第一次推廣，不一定要選最大版本。

能夠真實完成一次，比設計一份完美但一直沒有開始的教案更重要。
