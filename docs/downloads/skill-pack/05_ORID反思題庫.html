Skill Pack

ORID 反思題庫

使用目的

遊戲後反思是《我們的島》能否從「好玩」走向「有學習」的關鍵。

老師不需要一次問很多題。每次帶領時，從 O、R、I、D 各選一題即可。

如果遊戲中或討論中出現明顯情緒，R 層次可以多停留一點。先處理感受，再進入詮釋與行動，會比急著講道理更安全。

ORID 是什麼

  • O Objective 客觀：剛剛發生了什麼？
  • R Reflective 感受：我有什麼感覺？
  • I Interpretive 詮釋：這代表什麼？我怎麼理解？
  • D Decisional 行動：接下來我可以怎麼做？

通用版題庫

O 客觀

  • 遊戲中哪一句話或哪一幕讓你印象最深？
  • 哪一個角色的選擇讓你覺得意外？
  • 哪一回合開始，情勢有明顯改變？
  • 哪些話讓衝突變嚴重？哪些話讓對話變可能？

R 感受

  • 你在角色裡最強烈的感受是什麼？
  • 你有沒有一刻覺得委屈、生氣、無奈或被誤解？
  • 你聽到其他組發言時，有什麼感覺？
  • 遊戲結束時，你比較像鬆一口氣，還是還有話想說？
  • 如果這件事發生在真實生活中，你覺得被影響的人可能會有哪些情緒？

I 詮釋

  • 你覺得這場衝突是怎麼形成的？
  • 哪些話看起來像玩笑，卻可能造成傷害？
  • 為什麼有些人受傷了，卻不一定會立刻說出來？
  • 遊戲裡的橋，可能代表現實生活中的什麼？
  • 如果這件事發生在真實班級或學校，誰的聲音最容易被忽略？

D 行動

  • 下次如果你聽到類似的話，可以做一個什麼小行動？
  • 如果你不確定一句話會不會傷人，可以怎麼確認？
  • 我們班可以建立一個什麼提醒，讓彼此更被尊重？
  • 你希望自己下次在玩笑、起鬨或旁觀時，有什麼不同選擇？

國小版

O 客觀

  • 你記得遊戲裡誰遇到困難？
  • 哪一句話讓你覺得「咦，這樣說好像不太舒服」？
  • 哪一組做了讓你印象深刻的決定？

R 感受

  • 如果你是那個角色，你可能會有什麼心情？
  • 今天你有沒有替哪個角色擔心？
  • 你覺得哪一刻大家比較像在吵架？哪一刻比較像在合作？

I 詮釋

  • 為什麼有些玩笑會讓人難過？
  • 如果有人一直被同樣的話取笑，他可能會怎麼想？
  • 為什麼我們要先聽別人的感受？

D 行動

  • 下次開玩笑前，我可以先想什麼？
  • 如果我看到同學被取笑，我可以說什麼？
  • 我希望我們班多一個什麼樣的友善提醒？

國中版

O 客觀

  • 遊戲中哪個角色最常被誤解？
  • 哪些資訊出現後，你對事件的看法改變了？
  • 哪一組的選擇最影響結局？

R 感受

  • 你有沒有一刻覺得「這很不公平」？
  • 你扮演的角色有沒有想說卻不好說的話？
  • 當別組不理解你們時，你有什麼感覺？

I 詮釋

  • 「我只是開玩笑」和「對方真的受傷」之間，衝突在哪裡？
  • 為什麼少數人的聲音有時候比較不容易被聽見？
  • 如果一個人不是故意的，還需要為影響負責嗎？

D 行動

  • 下次如果朋友開了一個讓人不舒服的玩笑，你可以怎麼提醒？
  • 如果你發現自己說錯話，可以怎麼修復？
  • 班上要怎麼讓比較少說話的人也能安全表達？

高中版

O 客觀

  • 遊戲中哪些角色掌握較多資源或話語權？
  • 衝突是從個人言語開始，還是背後有更長的歷史與制度因素？
  • 哪些決策看似有效，卻可能忽略了受傷者的感受？

R 感受

  • 哪個角色讓你最難認同？為什麼？
  • 你是否曾經在遊戲中感覺自己被迫選邊？
  • 遊戲中是否有讓你想起現實生活的片段？

I 詮釋

  • 歧視一定來自惡意嗎？如果不是，還可以怎麼理解？
  • 「平等」和「公平」在遊戲中有什麼不同？
  • 為什麼只要求受傷的人說明自己的傷害，可能也是一種負擔？
  • 橋樑斷裂可以理解為關係斷裂，也可以理解為什麼社會現象？

D 行動

  • 如果校園裡出現類似微歧視，學生、旁觀者、老師可以分別做什麼？
  • 我們可以如何設計一個比較不依賴受害者主動求助的支持系統？
  • 你願意在日常中練習哪一種反歧視行動？

成人/教師研習版

O 客觀

  • 早上體驗中，哪一個遊戲機制最能引發討論？
  • 哪一個角色或隱喻最可能觸動你的學生？
  • 你觀察到自己作為玩家時，在哪些地方開始投入？

R 感受

  • 你在遊戲中曾經感到不確定、卡住或被挑戰嗎？
  • 如果換成你的學生，你最擔心他們在哪一刻被觸動？
  • 哪一段經驗讓你覺得「這可以帶回我的場域」？

I 詮釋

  • 你的學生最可能從這套桌遊學到什麼？
  • 你的學生最可能在哪裡誤解或失焦？
  • 你作為老師，最需要預先設計的是規則、情緒支持，還是反思提問？

D 行動

  • 你會先選輕量版、標準版，還是延伸版？為什麼？
  • 你第一個可執行場域在哪裡？
  • 你需要哪一種支持，才比較願意真的開始推廣？

選題提醒

  1. 每次不要問太多題。深問一題，比快問十題更好。
  2. 如果現場情緒高，先問 R，再慢慢進 I。
  3. 如果現場太抽象，回到 O，請學生描述具體事件。
  4. 如果學生只停在感動，最後要進 D，讓學習變成下一步行動。
