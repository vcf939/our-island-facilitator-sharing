# 《我們的島》教師推廣啟動包

## 這份技能包的目的

這份啟動包是給參加完《我們的島》反歧視桌遊工作坊的老師使用。

它不是補充閱讀資料，而是幫老師把「我覺得這套桌遊很有價值」轉成「我知道可以怎麼開始帶」。

## 個人分享與引用聲明

這份技能包由引導師 Vincent 個人整理，用於工作坊後的教師共學與推廣支持。

本技能包不是至善基金會官方文件，也不是官方授權教材。相關桌遊設定、議題脈絡與原始素材引用自至善基金會《我們的島》桌遊素材。

## 對應的推廣阻力

根據參與者成效追蹤報告，未執行者常見阻力包括：

- 時機尚未成熟：不知道可以放在哪一堂課、哪個場域使用。
- 操作覺得複雜：對流程、規則、判定與反思還不熟。
- 資源限制：時間、人力、備課量不足。
- 信心不足：擔心處理不了敏感對話、學生情緒、玩笑、價值衝突或沉默。

這份技能包用十個工具回應這些阻力。

## 技能包內容

1. `01_三種推廣版本.md`  
   幫老師選擇輕量版、標準版或延伸版，不必一開始就做到完整四堂課。

2. `02_一頁式教學設計單.md`  
   幫老師把自己的真實場域、對象、敏感情境與反思設計整理成一張可執行計畫。

3. `03_開場與安全聲明話術.md`  
   提供可以直接使用或改寫的開場語、安全約定與卸角語。

4. `04_敏感情境回應卡.md`  
   提供常見學生反應與老師回應範例，協助老師降低臨場焦慮。

5. `05_ORID反思題庫.md`  
   提供遊戲後反思題庫，並依國小、國中、高中與成人場域微調。

6. `06_第一步承諾卡與推廣回報.md`  
   幫老師把意願轉成下一步，也提供簡易回報與支持需求整理。

7. `07_校園歧視情境分類清單.md`  
   幫老師辨識校園裡常見的歧視情境，包含同儕、制度與日常微歧視。

8. `08_師長版情境卡.md`  
   協助老師看見師長或制度也可能在無意中造成歧視，並練習修復性的回應方式。

9. `09_技能包使用路徑.md`  
   說明哪些文件適合發給參加者、哪些適合引導師使用，以及不同時間長度的用法。

10. `10_歧視事件情緒地圖.md`  
   協助老師辨識被歧視者、造成傷害者與旁觀者在事件中的情緒落差。

11. `11_事件後副作用與修復檢核.md`  
   協助老師看見事件後可能擴散的副作用，並從受傷者、造成傷害者與群體規則三層進行修復。

12. `99_下午工作坊使用總表.md`  
   給引導師下午帶領時使用，可把這份啟動包嵌入共備流程。

## 下午工作坊建議用法

早上老師已經親自體驗桌遊，下午可以不用再重複大量說明遊戲精神。

下午的重點建議放在三件事：

1. 真實場景設計：老師找到一個自己真的可能使用的場域。
2. 敏感情境演練：老師練習遇到學生反應時如何接住與帶回學習。
3. 師長覺察：老師看見自身權力位置、善意偏誤與制度慣性。
4. 情緒與修復：老師判讀歧視事件中的情緒落差與事件後副作用。
5. 第一步承諾：老師選擇一個低門檻版本，寫下第一次試行計畫。

## 使用時的語氣提醒

談「師長造成的歧視」時，請避免把老師放在被審判的位置。

比較適合的語氣是：

- 我們都有可能在善意、效率、習慣或管理壓力中忽略學生感受。
- 重要的不是證明誰是壞人，而是看見影響，並練習修復。
- 因為師長有權力位置，所以無意的話語也可能被學生記得很久。
- 老師願意覺察與調整，本身就是推動反歧視教育的重要示範。

## 給老師的一句話

你不需要第一次就做到完美，也不需要成為歧視議題專家才可以開始。

你要做的是創造一個安全的練習場，讓學生透過遊戲看見：話語、選擇與行動，會如何影響人與人之間的橋。
