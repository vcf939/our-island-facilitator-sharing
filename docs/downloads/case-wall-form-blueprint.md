# 《我們的島》推廣案例牆投稿表

本表單用來收集老師願意整理成案例牆的推廣故事。  
案例牆的重點不是展示成果，而是讓其他老師看見實際帶領時的情境、卡點、調整與學習。

## 表單設定建議

- 表單名稱：《我們的島》推廣案例牆投稿表
- 表單說明：本表單為引導師 Vincent 個人共學整理使用，用於蒐集老師推廣《我們的島》反歧視桌遊後，願意整理成案例牆的現場故事。本表單不是至善基金會官方表單，也不是官方授權教材頁的一部分。請避免填寫可辨識學生個資。
- 確認訊息：謝謝您的案例分享。整理成案例牆前，建議會再確認匿名或具名方式，以及是否需要刪修可辨識資訊。
