推廣案例牆

這裡用來整理老師實際推廣後的故事。案例不只收成功，也收卡住、調整、修復與下一步，讓更多老師看見「原來可以這樣開始」。

案例牆為引導師個人共學用途，發布前建議取得分享者同意，並避免揭露可辨識學生個資。

案例格式

我在哪裡用？

場域、對象、時間、使用版本，以及為什麼選這個版本。

現場片段

學生最有反應的一刻

記錄一句學生說過的話，或一個讓老師印象深刻的互動。

調整學習

我下次會怎麼做？

包含遇到的風險、情緒處理、反思題調整與後續支持需求。

Case Template

案例撰寫格式

每一則案例建議控制在 600 到 900 字，讓老師可以快速讀完，也能抓到可以複製的做法。

案例標題

用一句話寫出場景與學習，例如：「國中導師用短版流程處理班級玩笑與貼標籤」。

1. 場域與對象

年段、場域、人數、時間長度、使用版本。若涉及敏感事件，請模糊化學校、班級、地點與個人資訊。

2. 為什麼想使用

老師當時想處理的班級需要、議題需要或課程目標。可以寫「想讓學生理解玩笑背後的傷害」這類真實動機。

3. 怎麼帶

簡述流程：開場安全聲明、遊戲體驗、沉澱提問、校園情境討論、行動承諾。也可註明使用了哪些技能包。

4. 關鍵片段

記錄一個最有反應的畫面或一句話。避免逐字揭露可辨識個資，可用「有學生提到...」改寫。

5. 卡點與處理

寫出現場困難，以及老師如何暫停、澄清、照顧情緒、重新設定對話規則或延後處理。

6. 帶走的學習

這次經驗讓老師對反歧視、霸凌預防、班級對話或教材設計有什麼新理解。

7. 給下一位老師的提醒

一句具體建議，例如：「不要急著講道理，先讓學生說出他看到的角色處境」。

Sample

範例案例摘要

一位國中導師在班會時間使用短版流程，先讓學生回想遊戲中「被排除」與「想加入」的感受，再轉到班上常見的綽號與玩笑。討論一開始有學生說「只是開玩笑」，老師沒有立刻否定，而是請同學分辨「說的人覺得好笑」與「聽的人感到被羞辱」之間的落差。最後，全班共同寫下三個新的互動提醒：不拿身體特徵取笑、不替別人決定他能不能接受、聽見有人說不舒服時先停下來。老師回饋，短版流程讓班級能先談感受，再談規則，對話比較不容易變成責備。
