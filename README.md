# 《我們的島》引導師共學分享

這個 repository 用來整理引導師 Vincent 個人對《我們的島》反歧視桌遊工作坊的共學分享、教師推廣技能包、回饋入口與案例牆。

本 repository 不是至善基金會官方網站，也不是官方授權教材頁。相關內容引用自至善基金會《我們的島》桌遊素材。

## 網站內容

GitHub Pages 網站放在 `docs/`：

- `docs/index.html`：共學站首頁
- `docs/feedback.html`：推廣回饋表建議欄位
- `docs/cases/`：推廣案例牆
- `docs/downloads/`：技能包下載檔
- `docs/assets/`：引導師簡介圖與網站素材

## 技能包

原始技能包放在：

`我們的島_教師推廣啟動包/`

網站下載版放在：

`docs/downloads/skill-pack/`

完整 ZIP：

`docs/downloads/我們的島_教師推廣啟動包_v1.0.zip`

## GitHub Pages 上線

請參考：

`docs/README_GitHub_Pages.md`

建議 GitHub Pages 設定：

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/docs`

## 回饋表

第一版建議先用 Google Form 收集老師推廣與技能包使用回饋。正式表單建立後，將 `docs/index.html` 的回饋按鈕替換成表單連結。

## 維護節奏建議

- 每月整理 1-3 則推廣案例。
- 每季檢視一次技能包是否需要更新。
- 技能包更新時，同步更新 `docs/downloads/skill-pack/` 與 ZIP 檔版本。
