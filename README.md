# 《我們的島》引導師共學分享

這個 repository 用來整理引導師 Vincent 個人對《我們的島》反歧視桌遊工作坊的共學分享、教師推廣技能包、回饋入口與案例牆。

本 repository 不是至善基金會官方網站，也不是官方授權教材頁。相關內容引用自至善基金會《我們的島》桌遊素材。

## 共學初衷

這兩年，我們在校園、職場與社會各處，看見不少令人憂心的霸凌事件。也因此，我更珍惜參與至善基金會推廣《我們的島》反歧視桌遊的經驗。

我希望把這份學習整理出來，透過「同理聆聽、浮現假設、平等對待」的對話練習，陪伴更多人看見歧視與霸凌如何發生，也一起練習如何修復關係、重新搭起理解的橋。

期待能與更多老師、社工，以及關心這個議題的公民一起共學，把桌遊中的體驗轉化為日常行動，促進更溫暖而正向的社會改變。

## 網站內容

GitHub Pages 網站放在 `docs/`：

- `docs/downloads/`：技能包下載檔

## 回饋表

## 維護節奏建議

- 每月整理 1-3 則推廣案例。
- 每季檢視一次技能包是否需要更新。
- 技能包更新時，同步更新 `docs/downloads/skill-pack/` 與 ZIP 檔版本。
